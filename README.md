# GKUOJ
程序设计竞赛在线判题系统，WEB端<a href = "https://github.com/chinawim/Judge" target="_blank" >（判题机端请点击）</a><br/>
# 说明
毕业设计<br/>
# 核心技术
前端：bootstrap4+vue+thymeleaf<br/>
后端：springboot+mybaits+springsecurity+springsession<br/>
数据库：mysql+redis<br/>
服务器：tomcat+nginx<br/>
消息队列：rocketMQ<br/>
